## 📊 Employee Churn Prediction Machine Learning Project

This repository contains an **Employee Churn Prediction** project developed as part of a machine learning course. The goal is to build and evaluate models that identify employees at risk of leaving the company, enabling proactive retention strategies.

### 🔍 Project Highlights

- **Data Sources**  
    - Data_phase1.csv – initial dataset  
    - train.csv & `test.csv` – later stages  
    - model_predictions.csv – output from model inference

- **Key Python Scripts & Notebooks**  
    - machine_project_2.py, `_v2`, `_improved`, and `_final` – progressively refined model implementations  
    - Kaggle-Project.ipynb – exploratory analysis, feature engineering, model training and evaluation  
    - Additional notebooks for experimentation (`Untitled93.ipynb`)

- **Approach**
    1. **Exploratory Data Analysis (EDA)** to uncover patterns and clean the data.
    2. **Feature engineering** to create meaningful predictors.
    3. **Modeling** using several classifiers (e.g. logistic regression, random forest, gradient boosting).
    4. **Evaluation** via metrics like accuracy, precision/recall, ROC‑AUC.
    5. **Prediction export** for reporting or integration with HR systems.

- **Purpose**
    - Demonstrate end‑to‑end ML workflow on a real‑world employee churn dataset.
    - Provide reproducible code and results to aid learning and collaboration.
